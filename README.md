# e-plantShopping

## Paradise Nursery Shopping Cart Application

e-plantShopping is a dynamic React and Redux based online plant shopping application developed for Paradise Nursery.

## Features

- Browse different houseplants
- View plant images, names, and prices
- Add plants to shopping cart
- Increase or decrease cart quantity
- Remove items from cart
- Dynamic cart item count
- Total cart price calculation
- Responsive and user-friendly interface

## Technologies Used

- React.js
- Redux Toolkit
- React Router DOM
- CSS

## Project Objective

The goal of this project is to build an interactive e-commerce shopping cart application for an online nursery shop.
